Módulo 02 - Exercícios 01 a 05

Exercício 01

Objetivo: Familiarizar-se com a estrutura condicional if/else.

Instrução: Escreva um programa que solicite ao usuário que insira um número. Se o número for maior que 10, o programa deve exibir "O número é maior que 10", caso contrário, "O número é menor ou igual a 10".

Exercício 02

Objetivo: Entender o uso de estruturas de repetição, especificamente o laço for.

Instrução: Crie um loop que exiba os números de 1 a 20 na console.

Exercício 03

Objetivo: Aprender a criar e usar funções.

Instrução: Crie uma função que aceite dois números como argumentos e retorne a soma desses dois números.

Exercício 04

Objetivo: Entender como os eventos funcionam no JavaScript.

Instrução: Crie um botão em HTML e adicione um ouvinte de evento que exiba um alerta na tela quando o botão for clicado.

Exercício 05

Objetivo: Familiarizar-se com a manipulação de JSON e o uso do LocalStorage.

Instrução: Crie um objeto JavaScript simples e converta-o em uma string JSON usando JSON.stringify(). Em seguida, salve essa string JSON no LocalStorage e, depois após atualizar a página, recupere e converta-a de volta em um objeto JavaScript usando JSON.parse().
